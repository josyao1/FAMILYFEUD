# Family Feud

A real-time Family Feud game you can run on any device. The host controls the game from their phone while the board displays on a TV or shared screen.

---

## How to Play

### Setup (takes ~1 minute)

1. **Host** — open the site on your phone and tap **Create New Game**. You'll be taken to the host control panel.
2. **Big screen** — on the TV or laptop, go to the same site, enter the 6-digit code shown on the host phone, and tap **Board →**. The game board will appear.
3. That's it. Everyone watches the big screen. Only the host needs to touch their phone.

---

## Game Flow

### Lobby
The host enters team names and player names for both teams, then taps **Start Game**.

### Face-Off
One player from each team faces off on the question. The host:
1. Taps the team that buzzed in first
2. Picks their answer from the list (or marks it as "not on the board")
3. Offers the other team a counter-guess
4. Decides **Play** (winning team plays the board) or **Pass** (losing team plays)

### Playing the Board
The playing team takes turns guessing answers. The host:
- Taps an answer tile to reveal it when a player guesses correctly
- Taps **✕ WRONG ANSWER** to add a strike (3 strikes = steal opportunity)
- Uses the **Next →** button to move to the next player

### Steal
After 3 strikes, the other team gets one guess to steal all the points. The host taps the matching answer tile (or marks it as not on the board), then awards points to the correct team.

### Round End
All answers are revealed. The host taps **Next Round** to continue or **End Game** to finish.

---

## Host Controls

| Button | What it does |
|--------|-------------|
| ⏱ 15s | Starts a 15-second countdown on the board |
| ✕ Ns / ✕ TIME | Dismisses the countdown |
| Roster ▼ | Edit player names mid-game |
| Skip Question | Pulls a new question during the face-off |

---

## Tips

- The board auto-syncs in real time — no refreshing needed.
- If someone opens the board mid-game they'll see the correct current state immediately.
- Sounds play on the board device, not the host phone.
- Questions are picked randomly from a pool of 1,800+. No question repeats in a single session.
- **2× multiplier** rounds double all points earned that round.
